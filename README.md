## Installation

Set up pip dependencies:

```
pip install -r requirements.txt
```

Set Environment Variable.

Windows:

```cmd
setx HOSHIKAWA_TOKEN "YOUR_TOKEN"
```

Linux:

```bash
export HOSHIKAWA_TOKEN="YOUR_TOKEN"
```

then edit src/setting.py to your config.

## Usage

```cmd
python src/main.py
```
